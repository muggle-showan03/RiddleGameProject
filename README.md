# 🧩 Enhanced Riddle Game

A professional Java-based riddle game built with **Object-Oriented Programming (OOP)** principles. This project was developed as part of my college coding practice to master Java Collections, Maven, and GitHub Actions.

## 🚀 Features
* **Randomized Gameplay**: Riddles are shuffled using `Collections.shuffle()` so every game is different.
* **Life System**: Players start with 3 lives. Lose them all, and it's Game Over!.
* **Hint System**: Stuck? You can ask for a hint at the cost of some points.
* **CI/CD Integrated**: Automatically builds and tests via GitHub Actions using Maven.

## 🛠️ Tech Stack
* **Language**: Java 17
* **Build Tool**: Maven
* **Concepts**: ArrayList, Generics, OOP (Classes & Objects), Scanner.
